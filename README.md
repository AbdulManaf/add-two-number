Add Two Number
=========

A small library that add two numbers

## Installation

  `npm install add-two-number`

## Usage

    var numAdditionar = require('add-two-number');

    var sum = numAdditionar(10,30);
  
  
  Output should be `40`


## Tests

  `npm test`

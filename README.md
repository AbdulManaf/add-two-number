Add Two Number
=========

A small library that add two numbers

## Installation

  `npm install @jdaudier/add-two-number`

## Usage

    var numAdditionar = require('@jdaudier/add-two-number');

    var formattedNum = numAdditionar(10,30);
  
  
  Output should be `40`
